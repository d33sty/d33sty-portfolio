# d33sty Portfolio

Моя персональная страница-портфолио со всеми ссылками и контактами, созданная на FastAPI.

## 🚀 Технологии

- **FastAPI** - современный веб-фреймворк для Python
- **Jinja2 Templates** - шаблонизатор для HTML
- **HTML5/CSS3** - стилизация и анимации
- **Uvicorn** - ASGI сервер для запуска

## ✨ Особенности

- 🎨 Современный дизайн с градиентами и анимациями
- 📱 Адаптивная верстка для всех устройств
- 🔗 Быстрый доступ ко всем социальным сетям
- 📋 Копирование номера телефона одним кликом
- 🎯 Интерактивные элементы при наведении

## 📋 Содержание

Страница содержит ссылки на:
- GitHub профиль
- Telegram канал
- Контактный телефон (с функцией копирования)

## 🛠 Установка и запуск

### Требования
- Python 3.7+
- pip

### Шаги для запуска

1. Клонировать репозиторий
```bash
git clone https://github.com/d33sty/d33sty-portfolio.git
cd d33sty-portfolio
```
2. Установить зависимости
```bash
pip install fastapi uvicorn jinja2

Или если используете requirements.txt:

pip install -r requirements.txt
```
3. Запустить приложение
```bash
uvicorn main:app --reload
```
4. Открыть в браузере
```bash
http://127.0.0.1:8000
```

## 🔧 Кастомизация

### Изменение ссылок
Отредактируйте templates/index.html и замените URL в секции .links:

<a href="ВАША_ССЫЛКА" target="_blank" class="link-card">
    <!-- ваш контент -->
</a>

### Изменение цвета фона
В templates/index.html найдите и измените градиент:

background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);

### Добавление новых кнопок
Скопируйте блок .link-card и настройте под свои нужды.
